# Extension Icons

The extension requires three icon files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

You can create these from the Nepal Airlines logo or use any 16x16, 48x48, and 128x128 PNG images.

The extension will work without these icons, but they won't display in the browser toolbar.
